## Sobre mim

Desenvolvedor Full Stack focado na construção de sistemas robustos, escaláveis e bem estruturados.

Atuo tanto no backend quanto no frontend, buscando sempre integrar as duas camadas de forma consistente e eficiente.

Tenho experiência com:

* Desenvolvimento backend com Node.js e TypeScript
* Arquitetura em camadas e boas práticas de engenharia de software
* Processamento assíncrono com filas e workers
* Desenvolvimento de interfaces modernas com frameworks atuais

Meu foco é criar soluções completas, que não apenas funcionem, mas que sejam sustentáveis, organizadas e preparadas para crescer.
